# ADR-0007: 家庭内网不管理防火墙

- **状态**: 已采纳
- **日期**: 2026-03-22
- **决策者**: @bernylinville

## 背景

NAS 在家庭内网 (192.168.50.0/24) 中运行，路由器提供 NAT 隔离。
nftables 作为系统工具已安装，但不配置规则。Docker 自身依赖 iptables/nftables 做端口映射。

## 决策

**不在 NAS 上配置防火墙规则**。nftables 仅作为 Docker 依赖保留。

## 理由

1. 家庭内网设备可信
2. Docker 默认网桥规则与防火墙规则容易冲突
3. 服务已绑定 LAN IP，不监听 `0.0.0.0`
4. 远程访问通过 EasyTier VPN，不走公网
5. 减少配置复杂度和维护成本

## 影响

- Ansible 不包含 nftables role（已移除）
- 安全依赖网络拓扑（路由器 NAT + VPN）而非主机防火墙
- 若未来接入不可信网络，需重新评估
