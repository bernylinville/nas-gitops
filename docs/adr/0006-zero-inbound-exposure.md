# ADR-0006: 零公网暴露安全策略

- **状态**: 已采纳
- **日期**: 2026-03-21
- **决策者**: @bernylinville

## 背景

NAS 存储个人数据，需要确保安全性。家庭网络通过路由器 NAT 隔离，
远程访问通过 EasyTier mesh VPN。

## 决策

所有服务**零公网暴露**：无端口转发、无公网 DNS、无 DDNS。

## 规则

1. 服务绑定 LAN IP (`192.168.50.10`)，禁止 `0.0.0.0`
2. 不在路由器上配置端口转发
3. 远程访问仅通过 EasyTier VPN
4. Docker 端口映射指定 IP，不使用默认
5. `.env` 文件权限 `0600`

## 影响

- 外部访问者无法直接触达 NAS
- compose 文件中端口绑定格式: `"${NAS_LAN_IP}:port:port"`
- 安全性以网络隔离为主，不依赖主机防火墙
