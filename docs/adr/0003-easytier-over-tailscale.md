# ADR-0003: 使用 EasyTier 远程接入

- **状态**: 已采纳
- **日期**: 2026-03-21
- **决策者**: @bernylinville

## 背景

需要从外部网络安全访问家庭 NAS，且不暴露公网端口。
路由器为 iStoreOS (192.168.50.1)，已部署 EasyTier。

## 决策

使用 **EasyTier** mesh VPN，由路由器代理，NAS 不直接运行 EasyTier 客户端。

## 备选方案

| 方案 | 优点 | 缺点 |
|------|------|------|
| **EasyTier (路由器代理)** | 已部署、NAS 零配置、路由器代理整个网段 | 依赖路由器稳定 |
| Tailscale | 成熟、客户端简单 | 依赖外部协调服务器 |
| WireGuard 手动 | 性能好、完全自控 | 配置复杂、密钥管理 |

## 影响

- NAS 无需安装 VPN 客户端
- 服务绑定 LAN IP (192.168.50.10) 即可通过 EasyTier 访问
- 若路由器方案不稳定，未来可考虑 NAS 端部署 (M4-E)
